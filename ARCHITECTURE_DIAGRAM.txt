║                         OFFLINE SYNC FLOW DIAGRAM                            ║
╚══════════════════════════════════════════════════════════════════════════════╝


    ┌─────────────┐
    │   User      │
    │  (Offline)  │
    └──────┬──────┘
           │
           │ Creates Farmer Data
           ▼
    ┌─────────────────────────┐
    │   SQLite Local DB       │
    │  syncStatus: PENDING    │
    └──────┬──────────────────┘
           │
           │ Background Task (Every 15 min)
           ▼
    ┌─────────────────────────┐
    │  Check Internet         │
    │  Connectivity           │
    └──────┬──────────────────┘
           │
           ├─── NO ──→ Wait for next cycle
           │
           │ YES
           ▼
    ┌─────────────────────────┐
    │  Send Pending Data      │
    │  to Server API          │
    └──────┬──────────────────┘
           │
           ▼
    ┌─────────────────────────┐
    │  Server Validates       │
    │  & Saves to MongoDB     │
    └──────┬──────────────────┘
           │
           ▼
    ┌─────────────────────────┐
    │  Check for Conflicts    │
    └──────┬──────────────────┘
           │
           ├─── Conflict? ──→ Apply Resolution Strategy
           │                  (LATEST_WINS / SERVER_WINS)
           │
           │ No Conflict
           ▼
    ┌─────────────────────────┐
    │  Update Local DB        │
    │  syncStatus: SYNCED     │
    └──────┬──────────────────┘
           │
           ▼
    ┌─────────────────────────┐
    │  Sync Complete!         │
    │  User Notified          │
    └─────────────────────────┘

ONLINE MODE:
─────────────

Mobile App  ──[1. Create Farmer]──→  Local SQLite
                                           │
                                           │ [2. Immediate Sync]
                                           ▼
                                      Backend API
                                           │
                                           │ [3. Validate & Save]
                                           ▼
                                      MongoDB Atlas
                                           │
                                           │ [4. Return Success]
                                           ▼
Mobile App  ←─[5. Update UI]─────  Local SQLite


OFFLINE MODE:
─────────────

Mobile App  ──[1. Create Farmer]──→  Local SQLite
                                      (syncStatus: PENDING)
                                           │
                                           │ [Wait for Internet]
                                           │
                                           │ [2. Internet Available]
                                           ▼
                                      Background Worker
                                           │
                                           │ [3. Batch Sync]
                                           ▼
                                      Backend API
                                           │
                                           │ [4. Process All Pending]
                                           ▼
                                      MongoDB Atlas
                                           │
                                           │ [5. Return Results]
                                           ▼
Mobile App  ←─[6. Update Status]──  Local SQLite
                                      (syncStatus: SYNCED)
